Unit 2

Unit 2 covered basic structure of an HTML document, along with covering some basic elements to start creating a meaningful document, with headings, paragraphs, lists, links, and tables. This provides a basis for a meaningful web page.

  • Head, with the HTML document declaration, meta tags, and title.
  • Paragraphs and headings, for getting text down on the page.
  • Links, both to files hosted on the website and to external sites across the internet.
  • Lists, to easily order and organize items.
  • Tables, to create a graphical depiction of information.
